Compiladores e IDEs

Os compiladores são ferramentas essenciais em qualquer linguagem de programação. Já as IDEs são editores de texto, feitos especialmente para desenvolvedores.


Incluindo Bibliotecas

Tal como em várias outras linguagens, em C, é essencial a inclusão de bibliotecas para que o código funcione. Neste artigo vamos te ensinar quais são as funções de algumas, e como incluí-las.


Função Main

Como o nome já diz, essa é a função principal. É nela onde acontece tudo o que você vê no programa.


Printf e Scanf

A função printf escreve uma mensagem na tela. Enquanto a função scanf recebe um valor digitado pelo usuário. Aprenda a utilizá-los aqui.


Variáveis

Uma das coisas mais importantes em todas as lingugens de programação, são as variáveis. Aprenda aqui como usá-las.


Operadores Matemáticos

O usuário digitou um valor. E agora? Com os operadores matemáticos, você pode fazer as operações matemáticas básicas com as variáveis.


If, Else e Else If

Quer exibir uma mensagem para o usuário caso ele digite uma coisa específica? Com os comandos if, else e else if, seus problemas acabaram.


Ciclo for

Precisa que algo em seu programa se repita sem que você precise escrever os mesmo comando várias vezes? Então o ciclo for é perfeito para você.


Comando system

Você alguma vez já usou o prompt de comando do Windows ou o terminal do Mac. O comando system te permite usá-los em seu programa.
